# Assessment project:
This is a basic assessment project that demonstrates the implementation of a simple TODO functionality using core programming concepts.
The main goal of this project is to showcase how to create, read, update, and delete TODO items in a minimalistic and functional way.
## Project Features:
1. Elevated Button,
2. floating Action Button
3. showcasing list Item
4. DialogBox,
5. Custom widget
